## 🚀 Cách chạy dự án

1. **Clone repository:**
   ```bash
   git clone https://github.com/dathtph53550/dathtph53550.git
   cd dathtph53550
   ```

2. **Cài đặt dependencies cho từng dự án:**
   ```bash
   cd [tên-dự-án]
   npm install
   ```

3. **Chạy dự án:**
   ```bash
   npx expo start
   ```
